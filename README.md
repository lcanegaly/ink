# MandelbrotFractal
Basic C++ OpenGL Mandelbrot Set Viewer  
Uses - GLFW, GLM, GLEW  
Generates fractal with basic escape time algorithm, writes directly to a texture.  
Pan and zoom using arrow keys, space, ctrl.
